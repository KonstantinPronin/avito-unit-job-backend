# avito-unit-job-backend

#### Short description:
   A tiny service designed to handle user balance.  
   Default port 8080.
   
#### How to run:
```shell script
    git clone https://github.com/KonstantinPronin/avito-unit-job-backend.git
    cd avito-unit-job-backend
    docker-compose build
    docker-compose up
```
